# Gilded Rose Refactoring Kata - Extended

This is a fork of https://github.com/emilybache/GildedRose-Refactoring-Kata

It introduces additional requirements to generate receipts so you can practice really hard-to-test (not just hard-to-understand) code refactoring. We extended the original Gilded Rose Kata for a unit-test, refactoring, and TDD training.

Currently the requirements are only provided in Russian.
